# Restaurant Web Application

## Description
A restaurant web application built using Spring Boot and microservices architecture. 
It implements JWT-based authentication for secure access and provides separate 
services for user management and order processing.


## Features
- JWT authentication
- Admin menu management
- Order service for managing orders
- Seperate admin and user functionality

  
## Tech Stack
- Spring Boot
- Microservices
- JWT
- MySQL
- order Service url-https://github.com/samarthjoshi2018-a11y/Orders

  
## Impact
- Simplifies menu management for administrators by allowing easy addition of items.
- Requires minimal input (image, image URL, and price) to add new menu items.
- Enhances the overall workflow and provides a smoother user experience.
