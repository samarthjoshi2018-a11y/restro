# Restaurant Web Application

## Description
A restaurant web application built using Spring Boot and microservices architecture. 
It implements JWT-based authentication for secure access and provides separate 
services for user management and order processing.


## Features
- JWT authentication
- Admin menu management
- Order service for managing orders

## Tech Stack
- Spring Boot
- Microservices
- JWT
- MySQL
